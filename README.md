# 👑 퀸카 - 카드 추천 서비스 (가명)

### 🏆 팀 효진 (가명)  

💳 **퀸카**는 당신에게 딱 맞는 카드를 추천해 주는 스마트한 서비스입니다!  
AI 기반으로 소비 패턴을 분석하여 **최고의 카드 조합**을 찾아드립니다.  

---

## 🎯 왜 퀸카인가?  
✅ **맞춤 추천** – 내 소비 패턴에 딱 맞는 카드 추천  
✅ **실시간 혜택** – 카드사별 최신 프로모션 제공  
✅ **데이터 기반 분석** – 마이데이터를 활용한 스마트 소비  

---

## 🚀 사용 방법  
1️⃣ **회원가입 및 카드 등록**  
2️⃣ **소비 내역 동기화**  
3️⃣ **맞춤 카드 추천 받기**  
4️⃣ **혜택 비교 후 선택**
